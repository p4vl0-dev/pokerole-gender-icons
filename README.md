# Pokerole Gender Icons

A simple module for Foundry VTT that shows gender icons (♂ ♀ ⚲) next to actors in the Actors directory when using the **Pokérole** system.

## Features

- Displays a small gender icon on the right side of each actor entry.
- Icons update automatically when you change an actor's gender.
- Works only if the system setting **"Enable Pokemon Gender Selector"** is turned on.
- Respects the `none` gender value (hides the icon).

## Manifest URL
```
https://raw.githubusercontent.com/p4vl0-dev/pokerole-gender-icons/refs/heads/main/module.json
```
## Usage

Once installed and enabled, open the **Actors** tab – you'll see gender icons next to each actor that has a gender assigned.

## Requirements

- Foundry VTT version 13
- [Pokérole](https://github.com/Pokerole-Software-Development/foundry-pokerole) game system
